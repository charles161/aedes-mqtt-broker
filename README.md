# Aedes MQTT Broker 

This image creates an MQTT Broker on your machine and allows for **tcp** and **wss** connections using Aedes and mqemitter-redis. 

## Libraries Used

- [Aedes]
- [mqemitter-redis]
- [websocket-stream]
